EVENT NAME: TREASURE HUNT




RULES :

  • Selected Finalists will be given a set of questions to be solved.
  • On completion of it, each team will get a clue that directs to a spot inside the college.
  • The same procedure will be repeated at each spot till your final spot.
  • The team which finishes it in minimum duration gets the first place.
  • If in case of no such team, the winner will be decided based on the total number of questions answered in each spot.
  • Max. Duration: 1 hour
